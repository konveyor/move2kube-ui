# Move2Kube-UI

An UI for interacting with [Move2Kube API](https://github.com/konveyor/move2kube-api).

## Usage

### Production

1. Run `make install`
1. Run `make build`
1. Run `make start`

### Development

1. Run `make install`
1. Run `make startdev`
